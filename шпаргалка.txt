Чтобы сделать один исполняемый .exe файл, мы используем PyInstaller. Вот пошаговая инструкция.
1. Установка PyInstaller
pip install pyinstaller

2. Подготовка: создай иконку (опционально)
Можешь создать файл icon.ico (размером 32x32 или 64x64) и положить в папку с проектом. Если иконка не нужна — пропусти.

3. Создание .exe файла
Перейди в папку с проектом и выполни команду:

Базовый вариант (консольное окно будет видно):
pyinstaller --onefile --name "CableParser" cable_master.py

Вариант без консольного окна (только графический интерфейс):
pyinstaller --onefile --noconsole --name "CableParser" cable_master.py

Полный вариант (с иконкой и без консоли):
pyinstaller --onefile --noconsole --icon=icon.ico --name "CableParser" cable_master.py

4. Где будет результат
После выполнения команды:

Папка build/ — временные файлы (можно удалить)
Папка dist/ — здесь будет твой CableParser.exe

5. Что нужно учесть
Проблема: JSON-файл не будет найден
Программа использует файл KKS_building_bounds.json. В .exe файле он не будет автоматически включён. 
Нужно положить KKS_building_bounds.json рядом с .exe файлом.
